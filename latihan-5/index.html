Deskripsi

Di atas adalah contoh penggunaan property display: grid; di CSS untuk membentuk Bento Layout atau sering dikenal juga dengan Bento Grid. Selain penggunaan display: grid;, contoh di atas juga menggunakan property box-shadow, border, border-radius, transition, dan pseudo :hover untuk setiap element menu/item bento.
